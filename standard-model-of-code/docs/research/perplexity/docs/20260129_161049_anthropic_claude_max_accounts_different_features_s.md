## Response

**Anthropic's Claude Max plan is a single tier with two usage options—**Max 5x** ($100/month) and **Max 20x** ($200/month)—both offering the same core features, including 5x or 20x more usage than Pro, higher output limits, conversation memory, early access to new models/features, and priority access during peak times[1][2][5][7].** These options differ only in usage multipliers (5x vs 20x Pro limits), not in features or model access, which remain identical across Max accounts[1][2][3][5].

### Key Features Shared by All Max Accounts
- **Usage and Limits**: Max 5x gives 5x more usage per session than Pro; Max 20x provides 20x. Both support larger inputs/outputs and extended context windows[1][2][5][7].
- **Memory and Access**: Conversation memory across sessions, priority during high demand, and early access to new models (e.g., Claude Opus 4.5) and tools like Claude for Chrome/Excel[1][2][4].
- **Additional Perks**: Access to all Claude models, third-party tool integration, and features like Claude Code in terminals (no extra cost beyond subscription)[1][3].
- No differences in support tiers, image generation (unavailable in any plan), or other capabilities—Max is positioned as the top consumer tier for heavy users[1][2].
