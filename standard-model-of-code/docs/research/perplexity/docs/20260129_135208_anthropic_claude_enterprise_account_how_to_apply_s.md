To apply for an **Anthropic Claude Enterprise account**, organizations must contact Anthropic's sales team directly, as there is no self-service sign-up process—unlike individual or Team plans.[5][6] This custom plan is designed for large-scale business use, including features like secure data integrations, admin controls, audit logs, and Claude Code bundling.[1][5]

### Sign-Up Process
1. Visit the official Claude pricing page at claude.com/pricing/enterprise or Anthropic's main site (anthropic.com), where the Enterprise section explicitly states **"Contact sales"** to initiate application.[5][7]
2. Submit a sales inquiry via the contact form, providing details about your organization, intended use cases (e.g., productivity gains, codebase integrations, or enterprise-scale AI), team size, and security needs.[5][6]
3. Anthropic's sales team reviews the application, which may involve additional verification for high-volume or enterprise use (typically 2-5 business days, longer for complex cases).[2]
4. Upon approval, access the Anthropic Console (console.anthropic.com) to configure billing, API keys, usage limits, and integrations. Enterprise options include invoice billing, granular spending controls, and scalability for intense usage.[1][2][5]

Note: General account creation (for API or individual use) starts at console.anthropic.com with email/Google sign-up and use-case description, but Enterprise requires sales escalation.[2][4]

### Business Requirements
- **Minimum scale**: Custom pricing, often cited as ~$60 per seat with a minimum of 70 users and 12-month contract (totaling ~$50,000 minimum), though exact terms are negotiated.[6]
- **Target organizations**: Large enterprises needing advanced features like company knowledge integration, GitHub connectivity, artifacts for collaborative workspaces, terminal-based coding agents, and 25-100% productivity boosts.[1][5]
- **Other needs**: Responsible AI usage commitment, payment method (credit card or invoice), and alignment with security policies. High-volume applicants face extra review.[2][5]
- Pricing is undisclosed publicly and varies; contact sales for quotes.[5][6]
