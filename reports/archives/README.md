## Report Naming Convention

`TOPIC_[DATE].md` for dated reports (e.g., `GAPS_ANALYSIS_2026-01-19.md`)
`TOPIC.md` for evergreen reports (e.g., `DESIGN_TOKEN_SYSTEM_AUDIT.md`)

## Adding a New Report

1. Create report file following naming convention
2. Include: Date, Author, Status, Classification
3. Add entry to this index and `docs/README.md`
