# Claude Code Enterprise & Team Plans for Developers

Claude offers **Team and Enterprise subscription tiers** designed for collaborative development work, with Claude Code access available at specific pricing levels.

## Team Plans

The **Team plan requires a minimum of 5 members** and comes in two seat types[4]:

- **Standard Seat**: $25/user/month (annual) or $30/user/month (month-to-month)[4]. This includes core collaboration features like chat, projects, increased usage capacity, admin controls, SSO, domain capture, centralized billing, and integrations with Microsoft 365 and Slack[4].

- **Premium Seat**: $150/user/month[4]. This adds **Claude Code access** and early access to new collaboration features, making it suitable for technical teams requiring development tools[4].

## Enterprise Plan

The **Enterprise tier has custom pricing** and includes governance features such as SSO, audit logs, fine-grained roles, and the largest context windows[1]. Enterprise pricing typically starts around $50,000 annually[6], though exact costs are negotiated based on organizational needs.

## Claude Code Availability

Claude Code is **not included in the Standard team seat**—it requires either a Premium team seat ($150/user/month) or an individual Pro subscription ($17/month annual, $20/month monthly)[1][4]. For teams using Claude Code at scale, the annual Team Standard seat cost of $300/year can pay for itself within a few months when distributed across multiple developers[7].
