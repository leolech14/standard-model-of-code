## The Ecosystem's Evolutionary Laws

Extracted from 9 months of project archaeology:

1. **Ambitious monoliths die, specialized successors live.** Central-MCP (everything-platform, 259K files) -> DEAD. OpenClaw (runtime), Refinery (dashboard), Elements (classification), Sentinel (automation) -> all ALIVE.

2. **Ideas migrate through death.** OKLCH survived 3 project deaths. "Semantic atoms" became Standard Model atoms. MCP patterns from Central-MCP became Elements MCP servers. The ideas are more durable than the projects.

3. **The first attempt names the problem, the second solves it.** ProfilePro named the problem (classify code for AI). Elements solved it. Central-MCP named the problem (ecosystem orchestration). OpenClaw + Refinery solved it.

4. **Classification impulse is constant.** Sept 2025: 11 biological organs. Oct 2025: 9 registries. Dec 2025: 167 atoms + 8 dimensions. The form changes; the impulse to classify everything doesn't.

5. **Growth is burst-mode, not continuous.** 73.7% of Elements commits in 10 days. YTPipe born in 50 minutes. REH born fully formed in one commit. This is creative eruption followed by consolidation.
