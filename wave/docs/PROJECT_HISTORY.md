## 4) Why This Architecture Exists Today

The current architecture is the direct consequence of five pressures:

1. **Prototype collapse pressure:** Spectrometer-era monoliths were powerful but brittle.
2. **Semantic precision pressure:** Role, Purpose, and Constraint needed explicit separation and recomposition.
3. **Scale pressure:** One engine could not both compute structure and manage global context effectively.
4. **Health pressure:** Static code metrics were insufficient; topology + flow + reachability + parity controls were required.
5. **Operational pressure:** Human-only maintenance did not scale, so agentic orchestration (autopilot, macro executor, post-commit refinement) became core.

In short: today’s `particle + wave + .agent` layout is not cosmetic. It is the accumulated solution to correctness, scale, epistemic rigor, and continuous architecture health.
