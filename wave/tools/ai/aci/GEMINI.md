# AI Context Intelligence (ACI)

> The "Brain" of the Standard Model. Processes raw code into refined, logistics-aware context.

## Component Roles

| File | Role | Description |
|------|------|-------------|
| `refinery.py` | **Refinement Engine** | Chunks code into `RefineryNode` parcels with Waybills. |
| `context_builder.py` | **Synthesis** | Combines parcels into high-order intelligence. |
| `research_orchestrator.py` | **Agent Flow** | Deep analysis logic. |

## Logistics Integration
The `Refinery` class is **Logistics-Aware**. It consumes `parent_parcel_id` and `batch_id` to maintain the chain of custody.

## Rules
- Every node produced MUST include a `waybill`.
- Maintain W3C PROV compliance in route events.
