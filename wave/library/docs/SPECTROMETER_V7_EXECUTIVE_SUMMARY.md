## ⚛️ **STANDARD MODEL DO CÓDIGO**

### **12 Quarks Fundamentais**
1. **BITS** - Unidades básicas de informação
2. **BYTES** - Grupos de bytes organizados
3. **PRIMITIVES** - Tipos de dados primitivos
4. **VARIABLES** - Armazenamento de dados
5. **EXPRESSIONS** - Computações e avaliações
6. **STATEMENTS** - Declarações executáveis
7. **CONTROL** - Estruturas de controle de fluxo
8. **FUNCTIONS** - Blocos de código reutilizáveis
9. **AGGREGATES** - Coleções de dados
10. **MODULES** - Unidades de organização
11. **FILES** - Arquivos de código
12. **EXECUTABLES** - Código executável

### **96 Hádrons Especializados**
Padrões arquitetônicos detectados incluindo:
- **Entities**, **DTOs**, **Repositories**
- **Services**, **Controllers**, **Handlers**
- **TestFunctions**, **Constructors**
- **APIHandlers**, **Middleware**

### **384 Sub-Hádrons HAIKU**
Classificação granular (4 por hadron):
- APIHandler → RESTController, GraphQLResolver, WebSocketHandler, WebhookReceiver
- Service → ApplicationService, DomainService, InfrastructureService, IntegrationService
