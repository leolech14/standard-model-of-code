The 200 Atoms

Standard Model of Code - Complete Structural Taxonomy

Atoms are the fundamental structural types of software. Every piece of code can be classified as one of 200 canonical atom types, organized into 4 phases (DATA, LOGIC, ORGANIZATION, EXECUTION) and 22 families.
